nb-repository:download

Full name:

org.netbeans.maven:nb-repository-plugin:1.4-SNAPSHOT:download

Description:

Goal for retrieving and expanding the lucene index of the given repository. That in turn is used by the populate goal.

Attributes:

  • Executes as an aggregator plugin.

Required Parameters

Name Type Since Description
nexusIndexDirectory File - location on disk where the index should be created. either empty or with existing index from same repository. then only update check will be performed.
User property is: nexusIndexDirectory.
repositoryUrl String - url of the repository to download index from. Please note that if you already have an existing index at nexusIndexDirectory, you should always use the same url for that directory.
User property is: repositoryUrl.
